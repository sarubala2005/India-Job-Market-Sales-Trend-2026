📄 India Job Market Sales Trend Analysis (Interactive Dashboard Creation using MS Excel)

🎯 Project Objective

The primary goal of this project is to analyze the 2026 Indian job market trends to help job seekers and recruiters understand salary benchmarks, hiring patterns, and the impact of work modes (Remote/Onsite). The project aims to identify which roles and locations offer the best financial growth.

What is the average high salary for an ML Engineer?
